## Usage

**To use as your global config**:
```bash

mv ~/.claude/CLAUDE.md ~/.claude/CLAUDE.md.backup

# Copy this template
cp planning-is-prompting/global/CLAUDE.md ~/.claude/CLAUDE.md

# Customize for your needs
# Edit ~/.claude/CLAUDE.md to add your preferences
```
